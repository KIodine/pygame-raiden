# pygame-raiden
A Raiden-like implementation by pygame.

Requirements:

    python 3.x,
    pygame 1.9.3
    
啟動步驟:

    有Python執行環境的啟動方式
    1.安裝Python 3+、Pygame
    2.下載所有檔案
    3.執行src/pyden.py
    
    沒有Python執行環境
    1.下載所有檔案
    2.執行src/pyden.exe
    *限制:僅限Windows作業系統
    
操控方式:

    WASD為上下左右，J為發射彈藥
